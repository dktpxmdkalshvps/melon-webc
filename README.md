## 📄 1. `melon_cli.py`

**초기 모델: Pandas 기반의 간결한 CLI 도구**

### 📝 프로젝트 소개

사용자로부터 직접 장르명을 입력받아 멜론의 특정 장르 Top 50 곡 정보를 터미널에 출력하는 기초 스크립트입니다.

### ✨ 핵심 기능

* **딕셔너리 매핑**: 8가지 주요 장르(발라드, 댄스 등)와 멜론 고유 코드를 직접 연결합니다.
* **Pandas 활용**: `pd.read_html`을 사용하여 웹 페이지의 테이블 데이터를 구조화된 데이터프레임으로 변환합니다.
* **데이터 필터링**: 수집된 전체 테이블 데이터 중 '곡정보' 컬럼의 상위 50개 항목만 추출합니다.

### 🛠 기술 스택

* **Language**: Python
* **Library**: `requests`, `BeautifulSoup4`, `pandas`

## 📄 2. `melon_cli_2.py`

**발전 모델: 동적 크롤링 및 함수형 구조 도입**

### 📝 프로젝트 소개

하드코딩된 장르 목록 대신, 프로그램 실행 시 멜론 사이트의 탭 메뉴를 크롤링하여 실시간으로 장르 목록과 URL을 확보하는 업그레이드 버전입니다.

### ✨ 핵심 기능

* **동적 URL 수집 (`get_urls`)**: 멜론 장르 메인 페이지의 HTML 구조를 분석해 현재 서비스 중인 장르 리스트를 자동으로 생성합니다.
* **모듈화**: URL 수집, 정보 추출, 랭킹 출력 등 로직을 함수 단위로 분리하여 가독성과 재사용성을 높였습니다.
* **데이터 정밀 추출**: `find_all("a")`를 통해 곡 제목과 가수를 각각 분리하여 수집합니다.

### 🛠 기술 스택

* **Language**: Python
* **Library**: `requests`, `BeautifulSoup4`

## 📄 3. `melon_gui.py`

**GUI 전환 모델: Tkinter 기반 데스크톱 앱**

### 📝 프로젝트 소개

터미널 환경에서 벗어나 사용자가 윈도우 창에서 편리하게 장르별 차트를 조회할 수 있도록 제작된 첫 번째 GUI 애플리케이션입니다.

### ✨ 핵심 기능

* **인터랙티브 UI**: `Combobox`를 통한 장르 선택과 버튼 클릭 기반의 직관적인 사용자 인터페이스를 제공합니다.
* **Treeview 위젯**: 순위와 곡 정보를 표(Table) 형태로 출력하여 가독성을 극대화했습니다.
* **사용자 피드백**: `messagebox`를 도입하여 장르 미선택이나 네트워크 오류 시 알림창을 띄워 사용자 경험을 개선했습니다.

### 🛠 기술 스택

* **Language**: Python
* **Library**: `tkinter`, `ttk`, `requests`, `pandas`, `BeautifulSoup4`

## 📄 4. `melon_gui_ver2.py`

**최종 하이브리드 모델: 실시간 동적 크롤링 GUI**

### 📝 프로젝트 소개

`melon_cli_2.py`의 **동적 크롤링 로직**과 `melon_gui.py`의 **GUI 환경**을 결합한 최종 완성형 프로젝트입니다. 웹사이트의 변경 사항에 유연하게 대응하면서도 강력한 사용성을 제공합니다.

### ✨ 핵심 기능

* **실시간 장르 인덱싱 연동**: 애플리케이션 실행과 동시에 최신 장르 탭 정보를 크롤링하여 콤보박스 목록을 실시간으로 업데이트합니다.
* **반응형 텍스트 출력**: `tk.Text` 위젯과 `fill=tk.BOTH` 설정을 통해 윈도우 크기에 따라 유동적으로 변하는 결과창을 제공합니다.
* **상세 헤더 설정**: 브라우저 정보 및 언어 설정 등 정밀한 `User-Agent` 헤더를 포함하여 크롤링 차단 가능성을 최소화했습니다.

### 🛠 기술 스택

* **Language**: Python
* **Library**: `tkinter`, `ttk`, `requests`, `BeautifulSoup4`
